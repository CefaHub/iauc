# IllitAuction

IllitAuction — аукцион для Paper/Spigot 1.20.1 с GUI (54 слота).  
Есть страничная навигация, сортировка, поиск, меню «Мои лоты», возврат лотов и покупка поштучно.  
Оплата идёт через экономику (IllitMoney как Vault-провайдер).

---

## Команды

### Игроки
- `/ah` — открыть общий аукцион
- `/ah sell` — выставить предмет на аукцион (берётся предмет из руки)
- `/ah search <текст>` — поиск по названию/типу предмета
- `/ah <ник>` — открыть аукцион конкретного игрока

### Админ-команды (требуется `illit.auction`)
Все админ-команды выполняются через `illit`:

- `/illit auction reload` — перезагрузить конфиг/сообщения (если включено в сборке)
- `/illit auction purge` — очистка/служебная команда (если включено в сборке)
- `/illit auction remove <id>` — удалить лот по ID (если включено в сборке)

---

## Покупка поштучно

При покупке открывается меню выбора количества. Управление:
- ЛКМ — +1
- Shift + ЛКМ — +16
- ПКМ — -1
- Shift + ПКМ — -16

Количество и итоговая стоимость отображаются в **лоре предмета** в меню выбора количества.

Ограничения:
- нельзя купить больше, чем осталось в лоте
- нельзя купить больше 64
- нельзя купить собственный лот (показывается меню-ошибка с барьером)

---

## Сортировка

Сортировка переключается ЛКМ по кнопке сортировки:
- сначала дешёвые
- сначала дорогие
- сначала новые
- сначала старые

## Админ-команды и пермишены

Админ-доступ ко всем админ-командам:
- `illit.auction`

Пример выдачи (LuckPerms):
- `lp group admin permission set illit.auction true`
