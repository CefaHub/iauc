# IllitAuction

Аукцион с GUI (54 слота), страницами, сортировкой, обновлением и меню «Мои лоты».
Лоты живут 48 часов, затем переносятся в «возвраты» (забираются в меню «Мои лоты»).

## Команды
- /ah — открыть аукцион
- /ah sell <цена_за_1> [кол-во] — выставить предмет из руки (до 64)
- /ah search <текст> — поиск по названию/материалу
- /ah <ник> — открыть аукцион игрока

## Админ-команды
- /illit ah reload
- /illit ah forceexpire
- /illit ah remove <listing_uuid>
- /illit ah money <set|add> <ник> <сумма> (временная внутренняя валюта)

## GUI
- Название: &f&lАукцион
- Lore: серый (&7), названия кнопок: белый (&f)
- Покупка поштучно:
  - ЛКМ: +1
  - Shift+ЛКМ: +16
  - ПКМ: -1
  - Shift+ПКМ: -16

## Сборка
Java 17 + Maven:
```bash
mvn -q clean package
```
Jar: target/IllitAuction-1.0.0.jar
